# The Data Hub

A repository of small data sets for conducting simple modelling, analysis, and pipeline architecture tests.

<br>

## GitHub Pages

Launch Sphinx:

```bash
mkdir docs
cd docs
sphinx-quickstart
```

Build:

```bash
sphinx-build -b html docs/source docs/build/html
```
